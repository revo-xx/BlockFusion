# BlockFusion
A professional JNI-based internal cheat for Minecraft 1.8.9.

### Features
- **Modern UI:** Clean and flat ImGui menu (Toggle with `INSERT`).
- **Combat:** KillAura (Advanced targeting), AutoClicker.
- **Movement:** Fly, Sprint, AutoJump.
- **Player:** Velocity (Anti-Knockback), NoFall.
- **SDK:** Custom C++ SDK for seamless interaction with Minecraft's JVM.

### Controls
- `INSERT`: Toggle Menu
- `END`: Safe Unload (Eject DLL)
- `F`: Toggle Fly
- `R`: Toggle KillAura
